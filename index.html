Unofficial UQM 0.7.0 Debian packages

Binaries of these packages are kept up-to-date with the current stable release of Debian, and with the current bi-annual and LTS releases of Ubuntu. Packages for older Debian and Ubuntu versions are archived here. For technical help with these packages, contact oldlaptop in #sc2 on Freenode. Thanks to Gekko for providing hosting.
